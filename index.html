Central Processing Unit (CPU)

The brain of the computer. It executes instructions and processes data at billions of cycles per second.

  • Cores: 16
  • Threads: 32
  • Clock: 5.2 GHz

Graphics Processing Unit (GPU)

Specialized for parallel processing. It renders images, video, and powers complex 3D simulations.

  • VRAM: 24GB
  • Cores: 16384
  • Ray Tracing: Yes

Random Access Memory (RAM)

High-speed temporary storage. It allows the CPU to access data quickly for active applications.

  • Capacity: 64GB
  • Type: DDR5
  • Speed: 6000 MT/s
